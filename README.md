# PHP REST API

Simple Rest Api built with PHP and MySQL.

## Features

-MVC structure
-CRUD operations
-MySQL database
-PDO connection
-Tested with Postman

## Endpoints

GET /?path=users
GET /?users&id=1
POST /?path= users&id
PUT /?path=users&id=1
DELETE /?path=users&id=1

## TECH stack

PHP
MySQL
PDO
Postman
